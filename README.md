# MDAT

Mdat make a directory and some python files for AtCoder.

## Install
```
$ git clone https://github.com/pheni-18/mdat.git
$ cd mdat
$ pip install -e .
```

## Usage
```
$ cd [working_dir]/
$ vim template.py
Write your template code
$ mdat [directory name]
```

### Example
```
$ cd atcoder/
$ vim template.py
def main():
    ans = 0
    print(ans)
$ mdat abc001
```


## Require
- Python 3.6
